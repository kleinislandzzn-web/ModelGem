# Model Gem

Model Gem 是一个面向 TikTok 内部跨团队协作的模型效果评测与选型 Demo。它把图像/视频模型做成“模型超市”：设计、产品、算法、开发和业务同学可以横向查看模型能力、已有评测结果、考点结论，并根据业务需求获得候选模型推荐。

详细交接说明见：

```bash
docs/ModelGem-飞书交接文档.md
docs/ModelGem-Feishu-Handoff.md
```

## 当前页面

- 🛒 模型超市：展示图像/视频模型、合规状态、能力标签、成本、性能、适合场景和限制。
- 🧪 评测题库：以题库入口方式沉淀已有评测资产，支持待整理题库。
- ⚖️ 横向对比：按题库切换，多 prompt × 多模型横向对比输出、评分和备注。
- ✨ 需求匹配：输入业务需求或飞书文档链接，解析重点考点并推荐候选模型。
- 🚀 一键试跑：输入 prompt，多选模型，统一参数试跑，并可将好题加入待整理题库。
- 🧭 接入指引：说明业务/设计师如何理解模型接入状态和使用边界。

## 本地运行

```bash
npm install
npm run dev
```

默认访问：

```bash
http://127.0.0.1:5173/
```

生产构建：

```bash
npm run build
```

## 交接包说明

压缩包建议包含：

- `src/`：网站源码。
- `public/`：模型封面和视频预览素材。
- `dist/`：已构建的静态产物。
- `docs/`：飞书交接文档。
- `package.json` / `package-lock.json`：依赖声明。
- `README.md`：快速启动说明。

不建议打包 `node_modules/`，接手同学运行 `npm install` 即可恢复依赖。
